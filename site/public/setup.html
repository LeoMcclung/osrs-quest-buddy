Setup

  1. Download the GitHub desktop application
  2. Run the downloaded installer (default settings should be fine)
  3. Open the GitHub application and login with your GitHub account
  4. Add the osrs-quest-buddy repository (already cloned locally)
    • After logging into the application it should ask to add local repositories, select it here (if it shows up)
    • Otherwise, click the "+" icon and select "add" and point to the directory where the repository was checked out on your computer
  5. Make changes to the homepage (index.html) file located in site/public directory
  6. View the changes in the GitHub application, and commit them to the master branch and sync with GitHub
    • Select change file(s) in the change list
    • Fill in commit summary and description(optional)
    • Click button "Commit to master"
    • After committing your changes, click the "Sync" button to push your local commit(s) to the remote GitHub repository
      This will trigger a build job (via Jenkins) and deploy the new contents of the site/ directory to the webserver!
  7. Visit old-school-runescape-quest-buddy.website to see your changes!
